# Tip: Le menu /harmony

Tapez `/harmony` pour accéder au menu interactif avec 30 commandes :

```bash
/harmony          # Menu complet
/harmony quick    # Validation rapide (~30s)
/harmony full     # Audit complet (~2-5 min)
```

Explorez le menu pour découvrir toutes les fonctionnalités.

> Ce tip ne s'affichera plus après cette session.
