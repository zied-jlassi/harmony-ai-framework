# Tip: Commencez chaque session par /go

La commande `/go` initialise votre session Harmony et affiche :

- La phase actuelle (1-5)
- La story active
- L'état du circuit breaker
- Les actions recommandées

```bash
/go
```

C'est votre point de départ pour chaque session de travail.

> Ce tip ne s'affichera plus après cette session.
