# Bienvenue dans Harmony Framework

**Harmony** est un framework d'IA qui **apprend et s'améliore** avec vous.

## Pourquoi Harmony ?

| Problème | Solution Harmony |
|----------|------------------|
| Claude oublie entre sessions | **Sentinel Memory** - Apprend de vos erreurs |
| Pas de structure projet | **5 Phases** - Discovery → Maintenance |
| Code sans contexte | **Guardian** - Exige une story avant de coder |
| Réinventer la roue | **42 agents** prêts à l'emploi |

## Auto-apprentissage

Harmony apprend de :
- Vos erreurs (error-journal.json)
- Vos patterns (learned-patterns.json)
- La communauté (profiles, specialties)

## Contribuer

Enrichissez le framework :
- Créez des **profiles** pour vos technos
- Partagez des **specialties** (gaming, security, AI...)
- Proposez des **agents** spécialisés
